# SignArchive
Sign Language Dictionary.

Allows user to manage signs (with clips, images or texts), meanings, words ect.

The sign language sometimes uses the same sign for different words/meanings and/or the same word/meaning have different signs for them in various regions and times. It was all taken into consideration while creating this app.

Made for offline use, but will be implemented into webpage.

![alt text](https://github.com/Hoinkas/SignArchive/blob/main/SignArchiveWordView.png?raw=true)

## Dev stack
Electron + React.js + Typescript

### To run
```
pnpm install
pnpm dev
```
